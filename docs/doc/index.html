4.) Getting started TOP

Copy pricing.css for example into your sites root/css folder:

  • root/css
    • pricing.css

add link tag into head tag

<doctype html>
<html>
    <head>
        ...
        <link rel="stylesheet" href="css/pricing.css">
        ....
    </head>
    <body>
        ....
    </body>
</html>

if you want support for ie8, copy respond.min.js for example into your sites root/js folder:

        ...
        <link rel="stylesheet" href="css/pricing.css">
        <!--[if IE 8]>
            <script type="text/javascript" src="assets/js/respond.min.js"></script>
        <![endif]-->  
        ....
